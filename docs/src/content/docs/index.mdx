## What You'll Learn

### 🎯 Essential Tasks
- **Dashboard Navigation**: Understanding statistics and key metrics
- **Volunteer Management**: Adding notes, managing profiles, and permissions
- **Shift Creation**: Setting up shifts with proper capacity and requirements
- **Signup Management**: Approving volunteers and managing waitlists

### 🔧 Advanced Features
- **Group Bookings**: Managing multi-volunteer group signups
- **Location Filtering**: Working with multi-location data
- **Reporting**: Generating insights from volunteer activity
- **API Integration**: Understanding external system connections

### 🚨 Problem Solving
- **Common Issues**: Quick fixes for typical problems
- **User Support**: Helping volunteers with their accounts
- **System Errors**: Troubleshooting technical problems

## Getting Help

:::tip[Need Immediate Assistance?]
For urgent issues or questions not covered in this documentation:

- **Technical Issues:** Check the [System Errors](/troubleshooting/system-errors/) guide

:::

## System Status Reference

Throughout this documentation, you'll see these status references:

- 🟢 **Active:** Feature is fully operational
- 🟡 **Pending:** Requires approval or action  
- 🔴 **Attention:** Needs immediate attention
- 🔵 **Info:** General information or tips
